Culture Quest (2024)

Description du projet :

Culture Quest est un RPG rogue like tour par tour, il suit une histoire classique s'inspirant de J-RPG cultes. Son système de combat s'inspire des jeux Pokemon et Dragon Quest, tout en ajoutant plusieurs éléments qui stimulent le cerveau, avec plusieurs types de questions sur les mathématiques, l'histoire, la géographie. Enchainez les combats et arrivez au boss final pour SAUVER la PLANÈTE avec votre CULTURE ! Le jeu est codé en IJava, un Java simplifié unique à l'IUT de Lille.

Compétences mise en œuvre :

    Savoir-faire :

  • Compétences algorithmiques
  • Diviser le travail pour mieux régner
  • Gestion et organisation des fichiers

    • Savoir-être :

  • Savoir communiquer
  • S'entraider
  • Persévérer

Labyrinth Story (2025)

Description du projet :

Labyrinth Story est un jeu de résolution de Labyrinthe, avec un générateur de labyrinthes aléatoires et un générateur de labyrinthes parfaits, une gestion de comptes, une gestion des touches et des musiques. Il y a un mode libre pour choisir son type de labyrinthe et sa longueur, un mode progression en 6 étapes avec des vues restreintes et limitées. Le jeu est codé en Java.

Compétences mise en œuvre :

    Savoir-faire :

  • Maîtriser le Java (POO, types, fonctions, packages, MVC)
  • Maîtriser l'IHM pour faire des interfaces
  • Maîtriser les graphes (Pour l'algorithme de recherche en profondeur)

    • Savoir-être :

  • Savoir communiquer
  • S'entraider
  • Consulter et se documenter en Java
  • Gérer les tâches entre les membres de l'équipe.

Coin Coin Quest (2025)

Description du projet :

Coin Coin Quest est un RPG rogue like dungeon crawler, qui s'inspire principalement de Dragon Quest III, pour son gameplay avec un système de classe, chaque classe à des compétences uniques avec des effets différents. Il y a un classement avec des scores, des boss séparés par miniboss séparés par des mobs. Le jeu est entièrement développé en Java dans un environnement de travail Agile. Il a une vue terminale complète et une gestion des touches.

Compétences mise en œuvre :

    Savoir-faire :

  • Maîtriser le langage Java (types, fonctions, packages, gestion des fichiers)
  • Diviser le travail pour mieux régner
  • Gestion et organisation des fichiers
  • Équilibrer les fonctionnalités

    • Savoir-être :

  • Savoir communiquer
  • S'entraider
  • Consulter et se documenter en Java
  • Gérer les tâches entre les membres de l'équipe.

Site Écomobilité Playstation (2024)

Description du projet :

Ce projet a pour objectif de mobiliser les compétences de développement web pour pouvoir réaliser un site web pour une filiale fictive d’éco-mobilité pour Playstation, comprenant un système de connexion, de paiement, de choix de covoiturage et de location. Nous étions limités à l’utilisation du HTML et du CSS, et nous disposions d’un savoir de base quant à l’utilisation de ces derniers et des cours correspondants.

Compétences mise en œuvre :

    Savoir-faire :

  • Création de pages web (HTML)
  • Stylisation des pages (CSS)
  • Gestion de fichiers et organisation

    • Savoir-être :

  • Savoir communiquer
  • Voir les difficultés de chacun pour favoriser l’entraide
